# eSIM Migration Guide

Document ID: DT_ESIM_002
Company: Deutsche Telekom
Department: Support
Region: Germany
Product: eSIM
Access Level: support_agent

## Activation Failure
When eSIM activation fails, support must confirm identity verification, remove expired QR codes, check device EID compatibility, and issue a fresh activation profile. The customer should connect to Wi-Fi before scanning the new QR code.

## Escalation
Escalate to SIM Provisioning L2 if EID binding fails, profile download repeatedly errors, or the activation server returns a permanent failure code.
